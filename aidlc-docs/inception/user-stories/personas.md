# User Personas

## Persona: The Terminal Gamer

**Name**: Alex  
**Role**: Casual terminal player  
**Technical Level**: Comfortable with terminal/CLI environments  

### Characteristics
- Wants a quick, fun distraction during work breaks
- Familiar with vim keybindings (uses them daily)
- Appreciates retro/ASCII aesthetics
- Values simplicity — doesn't want to read a manual to start playing
- Plays in short sessions (5-15 minutes)

### Motivations
- Quick entertainment without leaving the terminal
- Satisfying "just one more wave" gameplay loop
- Visual appeal of colored ASCII art in motion

### Frustrations
- Games that require complex setup or dependencies
- Slow or unresponsive controls
- Unclear visual feedback (can't tell what's happening on screen)
- No way to understand game mechanics without external docs

### Goals
- Start playing within seconds of launching
- Understand tower/enemy mechanics through intuitive visuals
- Feel rewarded for strategic tower placement
- Complete a session in under 15 minutes
